## Vim操作

vim主要模式就三种：

1. 默认模式，一般命令模式。可以复制粘贴文本
2. 编辑模式，按i进入，esc退出
3. 命令行模式，：/?任意一个进入，可以查找，替换，保存，退出，配置编辑器

vim只看不写还是挺爽的，操作快捷键极度方便。写的话虽然有自动补全，但是还是没ide的成熟的code editor舒服

### 操作

**光标移动**

```bash
n<space> # 跳到本行的第n个字符
n<Enter> # 下移n行

0 # 移动到行头
$ # 移动到行尾

G # 移动到最后一行
gg # 移动到第一行
:n # 移动到第n行
```

**查找**

```bash
/word #在当前行后第一个值为word的字符串
?word #在当前行之前第一个为word的字符串
# 一个往后找，一个往前找

n # 重复前一个查找操作
N # 反复重复前一个查找操作
```

**替换**

```bash
:<n1>,<n2>s/word1/word2/g # 在第n1行与n2行之间寻找word1这个字符串，并将该字符串替换为word2# 
# 比如：1，2s/is/IS/g,在1，2行的is替换成IS
:1,$s/word1/word2/g #将全文的word1替换为word2
:1,$s/word1/word2/gc：将全文的word1替换为word2，且在替换前要求用户确认。
```

**文本操作**

```bash
v #选中文本
d #删除选中的文本
dd #删除当前行
y #复制选中的文本
yy #复制当前行
p #将复制的数据在光标的下一行/下一个位置粘贴
u #撤销
Ctrl + r #取消撤销
大于号 > #将选中的文本整体向右缩进一次
小于号 < #将选中的文本整体向左缩进一次
```

**文本操作有神奇的组合键**

```
ggdG #删除所有的行
gg=G #格式化所有的行
11GddG$p #删除11行并粘贴到末尾
```

**编辑器配置**

```bash
:set paste # 设置成粘贴模式，取消代码自动缩进
:set nopaste # 取消粘贴模式，开启代码自动缩进
:set nu # 显示行号
:set nonu # 隐藏行号
:noh # 关闭查找关键词高亮
Ctrl + q # 当vim卡死时，可以取消当前正在执行的命令
```
